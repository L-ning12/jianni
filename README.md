# EduLearn - 现代课程学习平台

一个现代化的在线课程学习网站首页，采用纯静态HTML、Tailwind CSS和Font Awesome构建。

## 功能特点

- 现代响应式设计
- 流畅的动画效果
- 完整的页面结构
- 多种课程分类
- 专业讲师展示
- 学生评价系统
- 常见问题解答

## 技术栈

- HTML5
- Tailwind CSS v3
- Font Awesome 4.7.0
- 纯JavaScript

## 如何使用

1. 克隆仓库：git clone https://github.com/L-ning12/jianni.git
2. 打开 index.html 文件即可查看网站

## 页面结构

- 导航栏：包含网站logo、主导航链接、搜索框和登录/注册按钮
- 英雄区域：突出网站主题和价值主张
- 特色功能：展示平台的核心优势
- 课程分类：按类别展示课程
- 热门课程：展示推荐课程
- 学习路径：提供系统化的学习方案
- 讲师团队：展示专业讲师信息
- 学生评价：展示真实用户反馈
- 常见问题：解答用户疑问
- 行动召唤：鼓励用户注册
- 页脚：包含网站链接和联系信息
